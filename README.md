# Music Recommender with Streamlit and LLMs  

+ Streamlit으로 ChatBot 구현  
+ OpenAI, Gemini, Anthropic API 사용  
+ LangGrpah로 Agent와 Supervisor 구현  
+ Tavily API로 웹 검색   
+ Sub-Agents:  
  +  Web Search Agent  
  +  Load User's Preference Agent  

## Requirements  

+ langchain
+ langchain_community
+ langchain_openai
+ langchain_anthropic
+ langchain_google
+ langchain_tavily
+ streamlit
+ pandas
+ google-genai
